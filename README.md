# Smart Library Management System
 # Project Description
This project is a Library Management System developed using SQL.
It helps manage books, authors, members, and transactions in a library.

# Database Tables
The system consists of the following tables:


Authors – Stores author details


Books – Stores book information


Members – Stores library member details


Transactions – Stores borrowing and return records


Insert new records (Authors, Books, Members, Transactions)

Delete inactive members
